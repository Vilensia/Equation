План тестирования.

Представим наше уравнение в вииде a*x^2+b*x+c;

 Тест 1.

Проверка уравнение на линейность (коэффициент перед x^2 равен 0)
Исходные данные a = 0, b и c - любые значения
Ожидаем: Сообщение о том, что уравнение не квадратное, решение уравнения

Тест 2. 

Проверка, что уравнение не имеет корней
Исходные данные a = 8, b = 1, c = 12
Ожидаем: Сообщение, что уравнение не иммет корней

Тест 3.

Проверка, что уравнение имеет два одинаковых корня
Исходные данные a = 1, b = -2, c = 1
Ожидаем: Сообщение о том, что у уравнения два одинаковых корня, решение уравнения x = 1

Тест 4.

Проверка уравнения с двумя разными корнями

Исходные данные a = 1, b = -5, c = 6
Ожидаем: Решение уравнения, два корня x = 2 и x = 3
